# Method Pre-experiments

`study_type: method_pre_experiment`

Method code is isolated from phenomenon-characterization programs and results.
Repository-local output must be written below `results/pre_experiments/`.

Metrics containing predictions use aligned values for primary conclusions;
raw values and recovered scale are diagnostics. Pure GT baselines use raw data.
GT may be used for final evaluation, but not by future training-free selection
or refinement objectives.
